Tipos de formulários e atributos

tag <form>: contêiner que agrupa formulários informando um endereço para onde os dados serão enviados (post) ou recebidos(get)

input type=""

text number password email tel (numero de telefone) date color (rgb, hex, hsl) range (controle deslizante/slider) define valores numéricos fixos a partir de um valor mínimo e máximo. atributos do <input type="range"> acima:

value="50" (valor inicial =50)
min="50" (valor mínimo =50)
max="100" (valor máximo =100)
step="5" (pula de 5 em 5)

<input type="hidden"> (escondido)

checkbox

radio reset (botão de limpar formularios da pagina) button submit (envia os dados das seções do formulario para o endereço especificado na tag <form action="">) por meio do atributo action="" cada seção é uma caixa de entrada ou saída de dados inserida na tag <form>

outros

select (menu suspenso seleção de opções com determinados valores),
os valores de cada opção selecionável são determinados pelo atributo value=""

textarea

button

Atributos de formularios

placeholder <input> required <input> (campo obrigatório) min / max <input type="number"> (atrbui valor mínimo e máximo para uma caixa de entrada de números) minlength / maxlength <input type="text"> (atribui um valor mínimo e máximo de caracteres de uma caixa de texto) autofocus <input> (campo selecionado automaticamente quando aparecer na página) pattern <input type="text,password,email,tel"> (padrão de dados: [valor minimo-valor maximo]{quantidade de valores com esse padrão})
exemplo: 3 numeros de 0 a 9; 2 ou 3 letras de A a Z No pattern, o valor mínimo e máximo é uma sequência letras (a-z/A-Z) ou números (qualquer-qualquer)

novalidate <form novalidate> desativar a validação dados das seções do formulario da tag <form> no momento do envio o envio

method <form method="">
informa o método de acesso a dados pelos formulários:
method="get"(obter valor de um endereço ou servidor)
method="post" (enviar valor a um endereço ou servidor)
