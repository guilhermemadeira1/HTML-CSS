Formatação de código-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

A tag <code > </code >trasforma um texto comum em formato de código, ou seja, com todos as letras e caracteres com o mesmo tamanho.

Já a tag <pre > </pre > preserva os espaçamentos do texto, evitando que o texto fique numa só linha reta na página. .

Código python com identação:

          
              num= int(input('Digite um número'))
              if num % 2 == 0:
                print(f'O número {num} é par')
              else:
                print(f'O número {num} é ímpar')
              print('Fim fo programa')

No código acima usamos a tag <pre> </pre > para manter a identação do código, preservando os espaços.
E a tag <code > </code > para deixar o texto em formato de código.

Formatação de citações

Citações simples

Podemos fazer citações simples com a tag semântica <q > </q > que coloca aspas no texto atribuindo significado de citação dentro da página.
OBS: Simplesmente digitar aspas não dá significado de citação para o texo na página.
Exemplo: O computador é um burro muito rápido.

Citações completas

Podemos criar citações dentro de blocos de citações (Blockquotes) usando a tag semântica <blockquote> </blockquote>.
Exemplo:
Assim dizia Sócrates:

Só sei que nada sei.
Acima, colocamos a citação dentro de um blockquote (bloco de citação).
Também podemos atribuir, dentro da tag blockquote, o endereço de uma citação de uma matéria, livro ou página disponível na web. Usando o parâmetro:
<blockquote cite= "https://endereço.com.br/matéria" > </blockquote>

OBS: O parâmetro <blockquote cite= ""> apenas armazena o endereço na tag e será visível dentro do código HTML para algoritmos de busca.
