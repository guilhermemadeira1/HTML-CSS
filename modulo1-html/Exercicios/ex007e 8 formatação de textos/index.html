Principais formatações (Título h1)

Negrito/Destaque (Título h2)

Aqui temos uma palavra em NEGRITO (SEM SEMÂNTICA) usando a tag <b > palavra </b >
(<b > </b > é uma tag sem semantica pois apenas possui forma.

Strong (Título h3)

Já a tag < strong > < /strong >, além de colocar o termo em negrito, também atribui SEMÂNTICA/SIGNIFICADO ao termo, ou seja, não atribui apenas forma em negrito ao texto.

Itálico (Título h2)

Aqui temos um termo em ITÁLICO usando a tag <i > </i > SEM SEMÂNTICA.

Aqui temos um termo com ÊNFASE usando a tag <EM > </EM > atribuindo FORMA e SEMÂNTICA ao termo.

Texto marcado

Podemos criar também um texto marcado usando a tag <mark> </mark>
Temos outro texto marcado aqui usando css dentro da tag <mark> para aplicar cor a um intervalo de texto ESPECÍFICO.
Também pode aplicar estilo à TODAS OS TEXTOS MARCADOS aplicando CSS dentro de <head> </head > da seguinte maneira:

<head>
  <style>
      mark{"background-color: cor"}
  </style >
</head >

A marcação possui forma Amarela por padrão.

Texto pequeno e grande

<big> </big> texo grande(tag obsoleta).,
<small > </small > Texto pequeno(ainda não está obsoleta).

Texto deletado

Usa-se a tag SEMÂNTICA <del > </del >
Texto deletado significa que pode ser lido mas que deve ser deconsiderado pelo leitor.

Texto inserido

Usa-se a tag SEMÂNTICA <ins> </ins >
Texto inserido é o oposto do deletado, pois significa que o texto obrigatóriamente deve ser considerado pelo leitor.

Texto sublinhado (Título h3)

Também temos a tag NÂO SEMÂNTICA <u > </u >
Que apenas sublinha o texto, mas não tem valor semântico.
