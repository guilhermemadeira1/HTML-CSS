Criando uma lista ordenada <ol > </ol >, ao inserir mais itens, a lista vai alterar a numerção automaticamente.
Cada item da lista deve possuir a tag < li >, que significa list item.
O </li > no final do item é opcional.

Dentro da tag <ol > </ol > podemos inserir type="" para determinar o tipo da lista
e start="" para definir a numeração de partida.

A posição de início da lista start="" só pode ser definida em valores numéricos (posição 1,2,3...), mesmo se a lista for do tipo alfabética ou romano.

Tipos de lista ordenada

O type="1" cria lista numérica 1,2,3,4...
O type="A" cria lista alfabética maúscula A,B,C,D...
O type="a" cria uma lista alfabética minúscula a,b,c,d...
O type="I" cria lista em números romanos com letras maúsculas: I,II,III,IV...
O type="i" cria lista em números romanos com letras minúsculas i,ii,iii,iv...

Cascating Style Sheets, é uma linguagem de marcação para a criação do design do site.
